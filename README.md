# Personal Expense Tracker (Python)

## Description
A command-line based Personal Expense Tracker built using Python.
It allows users to add, edit, delete, and visualize expenses.
Data is saved and loaded using file handling (JSON).

## Features
- Add, edit, delete expenses
- Category-wise and date-wise summary
- Data persistence using JSON file handling
- Graphical visualization using Matplotlib

## Files
- expense_tracker.py → Main Python program
- expenses.json → Stores expense data
- output screenshots included

## How to Run
```bash
personal_expenses_tracker.py
